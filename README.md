# Sparse PCA Review Paper

## 📌 Overview
This project presents a comprehensive review of Sparse Principal Component Analysis (SPCA), an advanced dimensionality reduction technique that improves interpretability by introducing sparsity into principal components.

## 🎯 Objectives
- Understand limitations of traditional PCA
- Study SPCA methodology using Lasso & Elastic Net
- Compare SPCA with SCoTLASS and Thresholding
- Analyze real-world applications

## 📊 Key Features
- Sparse loadings for better interpretability
- Efficient optimization using elastic net
- Works well with high-dimensional datasets
- Improved feature selection

## 🧠 Applications
- Bioinformatics (Gene expression analysis)
- Image processing (Face recognition)
- Finance (Portfolio optimization)
- Machine Learning (Feature selection)
